# TopChurchPlus Test Matrix

最後更新：2026-06-09

本文件列出每次細部修正後建議的測試項目。不是每次都要全跑，但修改到某模組時，至少跑該模組的合法與非法流程。

## 共用檢查

每次任務至少確認：

- 相關中文檔案沒有 mojibake。
- `git diff --check` 通過。
- API JS 檔用 Node `--check`。
- Apps Script HTML partial 用 `new Function()` 解析。
- 若部署 API，測 `/health`。
- 若有中文 Demo 寫入，讀回確認中文可讀。
- 每月災難復原維護，執行 `.\tools\check-rebuild-readiness.cmd -RunSmoke`，確認本機工具、NAS API、GitHub、文件與 smoke test 可用，並人工確認 `docs/RECOVERY_SECRETS_CHECKLIST.md` 內的私密資料可取得。
- 全站 UI 樣式調整時，至少檢查登入入口、功能選單、清單表格、表單輸入框、Modal、系統開發管理頁籤是否可讀且沒有文字重疊。

## 登入與權限

合法流程：

- 使用 `the.king.of.sky@gmail.com` 登入。
- 功能卡片依權限顯示。
- 管理員/超級管理者能看應看系統。

非法流程：

- 隨便輸入帳密不可登入。
- 無權限功能不顯示，直接打 API 也需被拒絕。
- 登出後回登入頁，不白畫面。
- 閒置提示與自動登出正常。

## 系統開發管理

合法流程：

- 新增 Issue，類型、狀態、優先度與描述可保存。
- 文件 Review 可讀 `DISASTER_RECOVERY_REBUILD` 與 `RECOVERY_SECRETS_CHECKLIST`。
- 新增版本歷程時可填寫 Token 消耗，列表需顯示 Token 標籤。
- 未填 Token 消耗的版本歷程需顯示 Token 未記錄。

非法流程：

- 非超級管理者不可進入系統開發管理。
- 版本摘要未填不可保存。

## 專案管理

合法流程：

- 新增專案，填完整資料後保存。
- 編輯既有專案。
- 完全控制權限者可保存。
- 新增/編輯會議，與會者可全選與依部門快速勾選。
- 匯出 DOC/PDF。

非法流程：

- 必填未填不可保存。
- 未取得專案權限不可看敏感專案。
- 新專案尚未產生專案編號時，權限與會議功能不可造成錯誤。

## 財務管理

合法流程：

- 新增採購單。
- 專案採購可選專案編號。
- 上傳報價單 PDF。
- 建立預借單。
- 建立獨立請款。
- 請款單新增/編輯/印出。
- 支出證明新增/編輯/印出。
- 支出憑證類型可選一般或鐘點費。

非法流程：

- 必填未填不可保存。
- 支出證明不能沒有請款單。
- PDF 檔案不可讀時要有明確錯誤。
- 無權限不可編輯。

## 牧養管理

合法流程：

- 搜尋會友。
- 開啟會友詳細後清單收合。
- 新增/編輯會友。
- 上傳會友照片。
- 上傳新人資料圖片。
- 查看最近兩個月聚會狀況。
- 依牧養資料權限看到指定會堂資料。

非法流程：

- 同名資料保存時要警示。
- 無牧養會堂權限不可看該會堂會友。
- 牧區過長不可撐壞版面。
- 社區欄位不可錯顯小家完整結構。

## 表單系統

合法流程：

- 建立表單。
- 新增題目。
- 上傳圖片附件。
- 產生公開連結或短連結。
- 外部填寫必填 Email。
- 填寫後收到或產生編輯連結。
- 查看回覆統計。

非法流程：

- 停止表單後不可新增或編輯回覆。
- 過期表單不可新增或編輯回覆。
- 未填 Email 不可送出。
- 公開連結不可黑畫面或卡住。

## 櫃台工作台

合法流程：

- 以 PIN Code 進入志工櫃台。
- 顯示當前會堂。
- 可手動切換會堂。
- 處理報名繳費。
- 處理 QT 領取。
- QRCode 報到掃描可用。

非法流程：

- 無效 PIN 不可登入。
- 停用 PIN 不可登入。
- 攝影機未授權時要有可理解提示。
- 金流紀錄需帶會堂。

## QT 管理

合法流程：

- 查詢每月庫存。
- 新增庫存異動。
- 新增調撥。
- 查訂單與狀態。
- `/qt/stock-check` 能正確回傳可否下單。

非法流程：

- 庫存不足不可下單。
- 無效月份或會堂要有錯誤提示。
- 中文資料不可亂碼。

## 教育管理

合法流程：

- 查詢課程。
- 建立課程或班級。
- 查看學員課程狀態。
- 估算可開班人數。

非法流程：

- 未完成前一階段不可進下一階段。
- 無權限不可編輯課程資料。

## 資產管理

合法流程：

- 查詢資產清單。
- 欄位排序。
- 新增/編輯資產。
- 狀態顏色顯示。
- 從請款建立資產時保留來源。

非法流程：

- 無效位置不可保存。
- 無權限不可操作。

## 行政物資管理

合法流程：

- 查詢物資清單。
- 新增物資。
- 調整總庫存與分堂庫存。
- 新增庫存異動。

非法流程：

- 分堂庫存合計不可不合理。
- 數量不可為非法值。

## 場地預約

合法流程：

- 週檢視。
- 篩選可借用場地。
- 新增預約。
- 已借場地在該時段鎖定。

非法流程：

- 同場地時間重疊不可保存。
- 不可借用場地不可申請。
- Google Calendar 權限不足時需明確提示或走 NAS 替代流程。

## Zoom 帳號管理

合法流程：

- 週檢視。
- 篩選 Zoom 帳號。
- 新增借用。
- 不同時段可借同帳號。

非法流程：

- 同帳號同時段不可重疊。
- 結束時間不可早於開始時間。

## QRCode 活動管理

合法流程：

- 建立活動。
- 產生報到櫃台。
- 掃描會友 QRCode 報到。
- 內部查報到狀況。

非法流程：

- 非該活動 QRCode 不可報到。
- 重複報到需明確處理。

## 聚會統計

合法流程：

- 匯入或同步點名資料。
- 小家統計可查。
- 聚會統計可查。
- 會友詳細顯示最近兩個月狀況。

非法流程：

- 無權限不可看非負責會堂資料。
- 日期範圍錯誤需提示。

## 主日信息管理

合法流程：

- 秘書部或主任牧師可進入。
- 新增信息主題。
- 記錄講員與分享場合。
- 設定各會堂分享狀態。

非法流程：

- 非授權角色不可看。
- 非會堂場合不可被硬塞到會堂資料。

## Line App / LIFF

合法流程：

- Line App 管理系統可開啟。
- Channel 設定可保存。
- Rich Menu 可新增/編輯。
- LIFF config 可讀。
- 姓名 + 手機可綁定會友。
- 已綁定與未綁定提示分流。

非法流程：

- 缺 LINE ID Token 不可建立 session。
- Token aud 不符不可建立 session。
- Session 過期需要求重新開啟 Line App。
- LINE API 預設不可正式呼叫。
- LIFF security 強制模式未確認正式網域前不可開。

## 短連結

合法流程：

- 建立短連結。
- 外部打開可轉到目標。
- 表單公開連結可用短連結分享。

非法流程：

- 無效 URL 不可保存。
- 停用連結不可轉址。
